FuLang Documentation

Introduction

FuLang is a simple functional programming language designed. This document provides an overview of the language's features and syntax.

Running the Program

To start the program, navigate to the FuLang folder in the terminal and use one of the following commands:

  1. dotnet run {PathToFile} - Run the program with a specified file. Example: dotnet run ../program.fl

Math Operations

FuLang supports basic math operations including addition (+), subtraction (-), multiplication (*), division (/), and modulo division (%).

Variables

Variables can be assigned using the syntax ANYWORD = value. For example:

word = 10
a = 15

Printing

The print function is used to display values. For example:

print word
print a

Constants and Library Functions

FuLang includes the constants PI and E, as well as library functions such as sin, cos, sum, and echo.

Conditions and If-Else Statements

FuLang supports comparison operators (>, <, >=, <=, !=, ==) and if-else statements. The syntax for if-else statements is as follows:

| (condition) {...code...} - IF
\ {...code...} - ELSE

User Functions

User-defined functions are created using the syntax:

func functionName : arg1 arg2 ... argN = {
    ...code...
}

The return statement is used to return a value from a function.

Function Examples

Here are some examples of user-defined functions in FuLang:

Factorial

    func Fact : n = {
        | (n == 1) {return 1}
        \ {return (n * Fact(n - 1))}

    print Fact(10)
