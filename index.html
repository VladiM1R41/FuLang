Документация по FuLang

Введение

FuLang — это простой функциональный язык программирования, разработанный для образовательных целей и демонстрации основ функционального программирования. Он поддерживает основные математические операции, переменные, условия, функции и константы. В этом документе представлено подробное описание особенностей и синтаксиса языка.

Особенности

  • Поддержка базовых математических операций
  • Работа с переменными и константами
  • Библиотека встроенных функций
  • Условные операторы и конструкции if-else
  • Определение пользовательских функций

Рекомендации по установке

  1. Скачайте и разархивируйте проект FuLang или клонируйте репозиторий с исходным кодом.
  2. Перейдите в директорию проекта в терминале. Папка FuLang.
  3. Соберите проект с помощью командной строки:
    4. dotnet build
  4. Запустите проект с помощью команды dotnet run {PathToFile}, где {PathToFile} — путь к вашему файлу FuLang.
  5. Также можно добавить FuLang в переменную окружения PATH и запускать программы следующим образом:
    6. ./FuLang {PathToFile}

Запуск программы

Для запуска программы выполните следующие шаги:

  1. Перейдите в директорию с FuLang в терминале.
  2. Запустите программу с указанным файлом: dotnet run {PathToFile}

Пример:

dotnet run ../program.fl

Основной синтаксис

Математические операции

FuLang поддерживает основные математические операции, такие как:

  • Сложение: +
  • Вычитание: -
  • Умножение: *
  • Деление: /
  • Остаток от деления: %

Переменные

Переменные можно присваивать с помощью синтаксиса ЛЮБОЕ_СЛОВО = значение. Например:

word = 10
a = 15

Вывод значений

Для вывода значений используется функция print. Например:

print word
print a

Константы и библиотечные функции

FuLang включает в себя константы PI и E, а также библиотечные функции, такие как sin, cos, sum, и echo.

Условные операторы и конструкции if-else

FuLang поддерживает операторы сравнения (>, <, >=, <=, !=, ==) и конструкции if-else. Синтаксис:

| (условие) {...код...} - IF
\ {...код...} - ELSE

Пользовательские функции

Пользовательские функции создаются с помощью следующего синтаксиса:

func functionName : arg1 arg2 ... argN = {
    ...код...
}

Для возврата значения используется оператор return.

Примеры функций

Факториал

func Fact : n = {
    | (n == 1) {return 1}
    \ {return (n * Fact(n - 1))}
}

print Fact(10)

Фибоначчи

func Fib : num = {
    | (num <= 1) {return 0}
    \ { | (num == 2) {return 1}
        \ {return Fib(num - 1) + Fib(num - 2)}
    }
}

print Fib(12)

Минимум

func Min : a b = {
    | (a < b) return a
    \ return b
}

Максимум

func Max : a b = {
    | (a > b) return a
    \ return b
}

Наибольший общий делитель

func GCD : a b = {
    minNum = Min (a b)
    maxNum = Max (a b)
    | (minNum == 0) {return maxNum}
    \ { | (minNum == 1) {return 1}
        \ {return GCD (minNum, maxNum % minNum)}
    }
}

print GCD (10 4)

Структура системы

Проект представляет собой реализацию компилятора для функционального языка программирования "FuLang". Основные элементы системы включают:

Каталог FuLang

Файлы исходного кода:

  • AssignmentStatement.fs: Реализация операторов присваивания.
  • BinaryExpression.fs: Обработка бинарных выражений.
  • BlockStatement.fs: Работа с блоками операторов.
  • ConditionalExpression.fs: Реализация условных выражений.
  • ConstantExpression.fs: Обработка константных выражений.
  • Expression.fs: Базовый класс для всех выражений.
  • Function.fs: Определение и обработка функций.
  • FunctionalDefineStatement.fs: Определение функциональных операторов.
  • FunctionalExpression.fs: Обработка функциональных выражений.
  • FunctionalStatement.fs: Обработка операторов для функционального программирования.
  • Functions.fs: Реализация встроенных функций.
  • IfStatement.fs: Реализация условных операторов.
  • Lexer.fs: Лексический анализатор.
  • NumberExpression.fs: Обработка числовых выражений.
  • Parser.fs: Синтаксический анализатор.
  • PrintStatement.fs: Оператор вывода.
  • Program.fs: Главный модуль программы.
  • ReturnStatement.fs: Оператор возврата.
  • Statement.fs: Базовый класс для всех операторов.
  • Token.fs: Классы для работы с токенами.
  • TokenType.fs: Определение типов токенов.
  • UnaryExpression.fs: Обработка унарных выражений.
  • UserDefinedFunction.fs: Определение пользовательских функций.
  • Variables.fs: Работа с переменными.

Проектные файлы:

  • FuLang.fsproj: Файлы конфигурации проекта для компилятора.

Документация

  • README.md: Описание проекта, цели лабораторной работы и основные требования.

Пример программы

  • program.fl: Пример программы на языке FuLang, демонстрирующий основные возможности языка, такие как объявление переменных, математические операции, условия, функции и рекурсия.

Плюсы и минусы

Плюсы

  • Простота синтаксиса и лёгкость освоения.
  • Поддержка функционального программирования.
  • Наличие встроенных математических функций и констант.

Минусы

  • Ограниченная функциональность по сравнению с более распространёнными языками программирования.
  • Отсутствие поддержки сложных структур данных.
  • Ограниченные возможности для масштабируемости и расширяемости.

Масштабируемость и будущее развитие

Возможные направления будущего развития FuLang включают:

  • Добавление поддержки сложных структур данных, таких как списки и словари.
  • Улучшение системы ввода-вывода для работы с файлами и сетью.
  • Оптимизация производительности интерпретатора.
  • Разработка интеграции с другими языками программирования.
  • Расширение библиотеки встроенных функций и добавление новых математических и логических операций.
